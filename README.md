## アプリの概要

お客様がテーブルに座ったまま、店員を呼ばずにタブレットから注文できるシステムです。注文はリアルタイムでスタッフ側の画面に反映されます。

---

## 画面構成

### お客様側

- **TOP画面** - テーブル番号を確認し、注文を開始する
- **メニュー画面** - カテゴリーを選んでメニューを選択・カートに入れる
- **お会計画面** - 注文内容を確認し、割り勘計算・会計を依頼する
- **完了画面** - 会計処理完了後に表示

### スタッフ側

- **注文管理画面** - リアルタイムで注文を確認・提供済みフラグを管理

## 工夫した点

- **設計から実装まで一人で作成** - コードを書く前にExcelで画面設計・state設計・コンポーネント設計をまとめてから実装しました
- **ポカヨケ** - 注文履歴がない状態ではお会計ボタンを非活性化
- **リアルタイム同期** - onSnapshotでスタッフ画面とお客様画面を常に最新状態に保つ
- **テーブル管理** - occupied → finished → ready のステータス遷移で席の状態を管理
- **割り勘計算** - お会計画面で人数割り計算に対応
